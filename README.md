# Hotline-Miami-Promotional-Page
Página web creada con fines de práctica para la materia FRONT END

# REQUISITOS
Node Js
VS code
Live Server Extension
XAMPP

# 1. Clonar el repositorio
git clone https://github.com/Nelsoon056/Hotline-Miami-Promotional-Page.git

# 2. Instalar dependencias
npm install

# 3. Crear base de datos
crea una base de datos nueva en xampp y importa el archivo hotline_miami_db.sql

# 4. Uso
con xampp encendido ejecuta el siguiente comando en la consola: 

produccion: npm run start
testeo: npm run dev
